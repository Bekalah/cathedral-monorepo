## Deployment

### Cloudflare Pages Setup:
- Domain: `realms.cathedral144.pages.dev`
- Framework: Vite + Three.js
- Build: `npm run build`
- Output: `dist`

### Environment Variables:
```env
VITE_THREEJS_CDN=true
VITE_TEXTURE_PATH=/assets/textures/
VITE_AUDIO_PATH=/assets/audio/
```
